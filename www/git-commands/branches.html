Git: Branches

Basic branch workflow:

  • git branch – list branches
  • git switch -c feature/my-branch – create & switch
  • git switch feature/my-branch – switch to branch
  • git push --set-upstream origin feature/my-branch – publish new branch
  • git merge develop – merge develop into current branch
  • git branch -d feature/my-branch – delete local branch
